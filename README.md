# 💰 工资计算器 — 实时收入追踪 PWA

每一秒的努力都看得见。输入月薪，实时计算你的每秒收入，追踪工时、加班、休息，让时间价值一目了然。

## ✨ 功能

- **实时收入计算** — 输入月薪，每秒更新已赚金额，支持正常/加班双费率
- **工时追踪** — 工作、加班、休息三计时器，进度条直观展示
- **消费换算** — 添加想买的物品，查看「今天赚了 X 杯奶茶」的趣味换算
- **心愿单** — 设定大额目标，追踪距离实现还有多远
- **月进度环** — SVG 环形图展示本月收入占月薪比例
- **统计洞察** — Chart.js 收入趋势图、日均/最高/出勤天数统计
- **成就系统** — 全勤、加班王、万元户等 12 个成就徽章
- **下班报告** — 下班弹窗展示收入/工时明细，带金币爆炸动画
- **分享传播** — 生成精美 Canvas 收入卡片，一键分享到微信/朋友圈。包含收入百分位排名：「超过了全国 X% 的打工人」
- **数据管理** — CSV/JSON 导出导入，本地持久化存储
- **深色模式** — 自动跟随系统，也可手动切换
- **PWA 离线** — Service Worker 缓存，离线也可使用

## 🚀 快速开始

### 直接使用
用浏览器打开 `index.html` 即可，无需构建工具。

### 本地开发
```bash
# 任意 HTTP 服务器均可
python -m http.server 8080
# 或
npx http-server -p 8080
```

访问 `http://localhost:8080`。

## 🗺️ 使用流程

1. 设置月薪 → 自动计算每秒费率
2. 点击「🚀 上班」开始计时
3. 工时每秒累加，收入实时刷新
4. 可选「☕ 休息」暂停计时
5. 点击「🔚 下班」生成今日报告
6. 查看统计、成就，分享到朋友圈

## 🏛️ 技术栈

- **纯前端** — 单 HTML 文件（~4100 行），无框架依赖
- **Chart.js 4.4.7** — 收入趋势图表
- **Canvas 2D API** — 分享卡片生成（600×900 px）
- **Web Share API** — 移动端原生分享面板
- **localStorage** — 本地数据持久化
- **CSS 自定义属性** — 亮色/暗色双主题
- **PWA** — manifest.json + Service Worker 离线支持
